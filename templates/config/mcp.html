Creating Personal Access Tokens
For Jira Data Center:
  1. Log in to your Jira instance
  2. Go to your profile settings (click your avatar → Profile)
  3. Select "Personal Access Tokens" from the left menu
  4. Click "Create token"
  5. Enter a name for your token (e.g., "JIA Integration")
  6. Set an expiration date (optional but recommended)
  7. Click "Create" and copy the generated token
For Confluence Data Center:
  1. Log in to your Confluence instance
  2. Go to your profile settings (click your avatar → Profile)
  3. Select "Personal Access Tokens" from the left menu
  4. Click "Create token"
  5. Enter a name for your token (e.g., "JIA Integration")
  6. Set an expiration date (optional but recommended)
  7. Click "Create" and copy the generated token
Important: Copy the tokens immediately as they won't be shown again. Store them securely and never share them with others.
Required Permissions:
  • Read access to projects and issues (Jira)
  • Write access for creating/updating issues (if using creation features)
  • Read access to spaces and pages (Confluence)
  • Read access to user information
