Custom Headers Configuration
Common Header Examples:
OpenAI API:
{
  "Authorization": "Bearer sk-your-api-key-here",
  "Content-Type": "application/json"
}
Azure OpenAI:
{
  "api-key": "your-azure-api-key",
  "Content-Type": "application/json"
}
Anthropic Claude:
{
  "x-api-key": "your-anthropic-api-key",
  "Content-Type": "application/json",
  "anthropic-version": "2023-06-01"
}
Security Note: Your API keys are stored securely and never logged or shared.
Model Configuration Parameters
Common Parameters:
model

The AI model to use (e.g., "gpt-4", "gpt-3.5-turbo")

temperature

Controls randomness (0.0-2.0). Lower = more focused, Higher = more creative

max_tokens

Maximum response length (e.g., 2000)

timeout

Request timeout in seconds (e.g., 30)

top_p

Alternative to temperature (0.0-1.0)

frequency_penalty

Reduces repetition (-2.0 to 2.0)

Example Configuration:
{
  "model": "gpt-4",
  "temperature": 0.7,
  "max_tokens": 2000,
  "timeout": 30,
  "top_p": 1.0,
  "frequency_penalty": 0.0,
  "presence_penalty": 0.0
}
